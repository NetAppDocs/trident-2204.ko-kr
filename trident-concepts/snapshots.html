    ліјлҘЁ мҠӨлғ…мҲҸ

    08/02/2022 кё°м—¬мһҗ

    Astra Tridentк°Җ л“ңлқјмқҙлІ„лҘј мң„н•ң ліјлҘЁ мҠӨлғ…мғ· мғқм„ұмқ„ м–ҙл–»кІҢ мІҳлҰ¬н•ҳлҠ”м§Җ мһҗм„ёнһҲ м•Ңм•„ліҙмӢӯмӢңмҳӨ.

    • ONTAP-NAS, ONTAP-SAN, GCP-CV, Azure-NetApp-нҢҢмқј л“ңлқјмқҙлІ„мқҳ кІҪмҡ° к°Ғ PV(мҳҒкө¬ ліјлҘЁ)к°Җ FlexVolм—җ л§Өн•‘лҗ©лӢҲлӢӨ. л”°лқјм„ң ліјлҘЁ мҠӨлғ…мғ·мқҙ NetApp мҠӨлғ…мғ·мңјлЎң мғқм„ұлҗ©лӢҲлӢӨ. NetAppмқҳ мҠӨлғ…мғ· кё°мҲ мқҖ кІҪмҹҒ мҠӨлғ…мғ· кё°мҲ ліҙлӢӨ лӣ°м–ҙлӮң м•Ҳм •м„ұ, нҷ•мһҘм„ұ, ліөкө¬ к°ҖлҠҘм„ұ л°Ҹ м„ұлҠҘмқ„ м ңкіөн•©лӢҲлӢӨ. мқҙлҹ¬н•ң мҠӨлғ…мғ· ліөмӮ¬ліёмқҖ мғқм„ұн•ҳлҠ” лҚ° н•„мҡ”н•ң мӢңк°„кіј мҠӨнҶ лҰ¬м§Җ кіөк°„ лӘЁл‘җм—җм„ң л§Өмҡ° нҡЁмңЁм Ғмһ…лӢҲлӢӨ.

    • ONTAP-NAS-Flexgroup л“ңлқјмқҙлІ„мқҳ кІҪмҡ° к°Ғ PV(Persistent Volume)к°Җ FlexGroupм—җ л§Өн•‘лҗ©лӢҲлӢӨ. л”°лқјм„ң ліјлҘЁ мҠӨлғ…мғ·мқҙ NetApp FlexGroup мҠӨлғ…мғ·мңјлЎң мғқм„ұлҗ©лӢҲлӢӨ. NetAppмқҳ мҠӨлғ…мғ· кё°мҲ мқҖ кІҪмҹҒ мҠӨлғ…мғ· кё°мҲ ліҙлӢӨ лӣ°м–ҙлӮң м•Ҳм •м„ұ, нҷ•мһҘм„ұ, ліөкө¬ к°ҖлҠҘм„ұ л°Ҹ м„ұлҠҘмқ„ м ңкіөн•©лӢҲлӢӨ. мқҙлҹ¬н•ң мҠӨлғ…мғ· ліөмӮ¬ліёмқҖ мғқм„ұн•ҳлҠ” лҚ° н•„мҡ”н•ң мӢңк°„кіј мҠӨнҶ лҰ¬м§Җ кіөк°„ лӘЁл‘җм—җм„ң л§Өмҡ° нҡЁмңЁм Ғмһ…лӢҲлӢӨ.

    • 'ONTAP-SAN-мқҙмҪ”л…ёлҜё' л“ңлқјмқҙлІ„мқҳ кІҪмҡ° PVSлҠ” кіөмң лҗң FlexVolм—җ л§Өн•‘лҗ©лӢҲлӢӨ. PVSмқҳ ліјлҘЁ мҠӨлғ…мғ·мқҖ м—°кІ°лҗң LUNмқҳ FlexCloneмқ„ мҲҳн–үн•ҳм—¬ кө¬нҳ„лҗ©лӢҲлӢӨ. ONTAPмқҳ FlexClone кё°мҲ мқ„ мӮ¬мҡ©н•ҳл©ҙ мөңлҢҖ к·ңлӘЁмқҳ лҚ°мқҙн„° м„ёнҠёмқҳ ліөмӮ¬ліёмқ„ кұ°мқҳ мҰүмӢң мғқм„ұн•  мҲҳ мһҲмҠөлӢҲлӢӨ. ліөмӮ¬ліёмқҖ лҚ°мқҙн„° лё”лЎқмқ„ л¶ҖлӘЁлӢҳкіј кіөмң н•ҳлҜҖлЎң л©”нғҖлҚ°мқҙн„°м—җ н•„мҡ”н•ң кІғл§Ң л№јкі  мҠӨнҶ лҰ¬м§ҖлҘј мӮ¬мҡ©н•ҳм§Җ м•ҠмҠөлӢҲлӢӨ.

    • 'мҶ”лқјмқҙл“ңмӮ°' л“ңлқјмқҙлІ„мқҳ кІҪмҡ°, к°Ғ PVлҠ” NetApp Element мҶҢн”„нҠёмӣЁм–ҙ/NetApp HCI нҒҙлҹ¬мҠӨн„°м—җм„ң мғқм„ұлҗң LUNм—җ л§Өн•‘лҗ©лӢҲлӢӨ. ліјлҘЁ мҠӨлғ…мғ·мқҖ кё°ліё LUNмқҳ мҡ”мҶҢ мҠӨлғ…мғ·мңјлЎң н‘ңмӢңлҗ©лӢҲлӢӨ. мқҙлҹ¬н•ң мҠӨлғ…мғ·мқҖ нҠ№м • мӢңм җ ліөм ңліёмқҙл©° мҶҢлҹүмқҳ мӢңмҠӨн…ң лҰ¬мҶҢмҠӨмҷҖ кіөк°„л§Ң м°Ём§Җн•©лӢҲлӢӨ.

    • ONTAP-NASмҷҖ ONTAP-SAN л“ңлқјмқҙлІ„лҘј н•Ёк»ҳ мӮ¬мҡ©н•  кІҪмҡ° ONTAP мҠӨлғ…мғ·мқҖ FlexVolмқҳ мӢңм җ ліөмӮ¬ліёмңјлЎң FlexVol мһҗмІҙмқҳ кіөк°„мқ„ мӮ¬мҡ©н•©лӢҲлӢӨ. л”°лқјм„ң мҠӨлғ…мғ·мқҙ мғқм„ұ/мҳҲм•Ҫлҗ  л•Ң мӢңк°„мқҙ кІҪкіјн•Ём—җ л”°лқј ліјлҘЁмқҳ м“°кё° к°ҖлҠҘн•ң кіөк°„мқҙ мӨ„м–ҙл“Ө мҲҳ мһҲмҠөлӢҲлӢӨ. мқҙ л¬ём ңлҘј н•ҙкІ°н•ҳлҠ” к°„лӢЁн•ң л°©лІ• мӨ‘ н•ҳлӮҳлҠ” KubernetesлҘј нҶөн•ҙ нҒ¬кё°лҘј мЎ°м •н•ҳм—¬ ліјлҘЁмқ„ лҠҳлҰ¬лҠ” кІғмһ…лӢҲлӢӨ. лҳҗ лӢӨлҘё мҳөм…ҳмқҖ лҚ” мқҙмғҒ н•„мҡ”н•ҳм§Җ м•ҠмқҖ мҠӨлғ…мғ·мқ„ мӮӯм ңн•ҳлҠ” кІғмһ…лӢҲлӢӨ. KubernetesлҘј нҶөн•ҙ мғқм„ұлҗң VolumeSnapshotмқҙ мӮӯм ңлҗҳл©ҙ Astra Tridentк°Җ м—°кІ°лҗң ONTAP мҠӨлғ…мғ·мқ„ мӮӯм ңн•©лӢҲлӢӨ. KubernetesлҘј нҶөн•ҙ мғқм„ұлҗҳм§Җ м•ҠмқҖ ONTAP мҠӨлғ…мғ·лҸ„ мӮӯм ңн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    Astra TridentлҘј мӮ¬мҡ©н•ҳл©ҙ VolumeSnapshotsмқ„ мӮ¬мҡ©н•ҳм—¬ мғҲ PVSлҘј мғқм„ұн•  мҲҳ мһҲмҠөлӢҲлӢӨ. мқҙлҹ¬н•ң мҠӨлғ…мғ·м—җм„ң PVSлҘј мғқм„ұн•ҳлҠ” мһ‘м—…мқҖ м§ҖмӣҗлҗҳлҠ” ONTAP л°Ҹ CVS л°ұм—”л“ңм—җ FlexClone кё°мҲ мқ„ мӮ¬мҡ©н•ҳм—¬ мҲҳн–үлҗ©лӢҲлӢӨ. мҠӨлғ…мғ·мңјлЎңл¶Җн„° PVлҘј мғқм„ұн•  л•Ң, л°ұм—… ліјлҘЁмқҖ мҠӨлғ…мғ·мқҳ мғҒмң„ ліјлҘЁмқҳ FlexCloneмһ…лӢҲлӢӨ. 'мҶ”л“ңнҢҢмқҙм–ҙ-SAN' л“ңлқјмқҙлІ„лҠ” Element мҶҢн”„нҠёмӣЁм–ҙ ліјлҘЁ нҒҙлЎ мқ„ мӮ¬мҡ©н•ҳм—¬ мҠӨлғ…мғ·м—җм„ң PVSлҘј мғқм„ұн•©лӢҲлӢӨ. м—¬кё°м„ң Element мҠӨлғ…мғ·мңјлЎңл¶Җн„° нҒҙлЎ мқ„ мғқм„ұн•©лӢҲлӢӨ.
